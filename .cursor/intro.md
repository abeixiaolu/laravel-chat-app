## 项目简介

一个基于 Laravel 和 React 的聊天App，支持聊天基础功能，以及群组功能（创建、加入、退出）、聊天附件功能。

## 模型

- 用户 User
- 群组 Group
- 消息 Message
- 附件 MessageAttachment
- 会话 Conversation

## 技术栈

### 前端

- React
- TailwindCSS
- @headlessui/react
- @heroicons/react
- daisyui
- emoji-picker-react
- react-markdown
- uuid

### 后端

- PHP
- Laravel
- Breeze
- Eloquent
